## Supabase для real-time дашбордов

### Как работает Realtime

```
PostgreSQL WAL → Logical Decoding → Realtime Server (Elixir) → Phoenix Channels → WebSocket → Client
```

Три механизма:
- **Postgres Changes** — подписка на INSERT/UPDATE/DELETE
- **Broadcast** — сообщения между клиентами
- **Presence** — онлайн-пользователи

### Free Tier

| Ресурс | Лимит |
|--------|-------|
| Database | 500 MB |
| File storage | 1 GB |
| Realtime connections | 200 concurrent |
| MAU | 50,000 |
| Проекты | 2 |
| Пауза | через 1 неделю неактивности |

### Паттерн: n8n → Supabase → Dashboard

```
n8n Schedule → HTTP Request (Avito/Bitrix) → Supabase Node (INSERT)
    → PostgreSQL WAL → Realtime Server → WebSocket → Next.js → UI update
```

В n8n есть встроенная **Supabase нода** (CRUD + Get All).

### Код подписки (Next.js)

```typescript
'use client'
import { createClient } from '@supabase/supabase-js'

const supabase = createClient(URL, ANON_KEY)

useEffect(() => {
  // Начальная загрузка
  supabase.from('listings').select('*').then(({ data }) => setListings(data))

  // Realtime подписка
  const channel = supabase
    .channel('changes')
    .on('postgres_changes',
      { event: 'INSERT', schema: 'public', table: 'listings' },
      (payload) => setListings(prev => [payload.new, ...prev])
    )
    .subscribe()

  return () => supabase.removeChannel(channel)
}, [])
```

## AI-powered дашборды

### Три уровня AI

**Уровень 1: AI Insights** (автоанализ)
```
n8n Schedule → Read Supabase → OpenRouter (llama-3.3-70b free) → Telegram + Dashboard
```

**Уровень 2: Text-to-SQL** (ad-hoc запросы)
- Пользователь спрашивает на русском → LLM генерирует SQL → результат + график
- **Vanna.AI** (12K+ stars, MIT): https://github.com/vanna-ai/vanna

```python
vn.train(ddl="CREATE TABLE objects (id INT, city TEXT, price INT, status TEXT)")
vn.ask("Топ-5 самых дорогих объектов в Калининграде")
# → SQL + DataFrame + Plotly chart
```

**Уровень 3: Anomaly Detection** (мониторинг KPI)
- Автоматическое обнаружение аномалий в метриках
- Минимизация false alerts

### AI Insights через OpenRouter (пример)

```typescript
// api/insights/route.ts
const response = await fetch('https://openrouter.ai/api/v1/chat/completions', {

  })
})
```
