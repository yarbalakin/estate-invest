## 1. ФИНАНСОВЫЕ МЕТРИКИ

### Форматирование чисел

**Правило: сокращай, не перегружай.**

| Диапазон | Формат | Пример |
|----------|--------|--------|
| < 1 000 | как есть | `750` |
| 1K - 999K | `XXX.XK` | `196K` |
| 1M - 999M | `XXX.XM` | `196M` |
| >= 1B | `X.XB` | `1.2B` |

Для рублей: `196 млн ₽` или `196M ₽`. В таблицах — полные числа: `196 000 000 ₽`.

```javascript
function formatNumber(num) {
  if (Math.abs(num) >= 1e9) return (num / 1e9).toFixed(1) + 'B';
  if (Math.abs(num) >= 1e6) return (num / 1e6).toFixed(1) + 'M';
  if (Math.abs(num) >= 1e3) return (num / 1e3).toFixed(1) + 'K';
  return num.toString();
}
```

### Цветовое кодирование

```css
:root {
  --color-profit:  #22c55e;  /* зелёный — прибыль, рост */
  --color-loss:    #ef4444;  /* красный — убыток, падение */
  --color-neutral: #94a3b8;  /* серый — без изменений */
  --color-warning: #f59e0b;  /* янтарный — внимание */
  --color-info:    #3b82f6;  /* синий — информация */
}
```

**Для дальтоников (8% мужчин):** всегда дублировать цвет иконкой (▲/▼) или текстом (+12%, -5%).

### Паттерн KPI-карточки

```
┌─────────────────────┐
│  Revenue        ↑   │  ← название + тренд-иконка
│  196M ₽             │  ← основное значение (крупный шрифт)
│  +12.3%  vs Q3      │  ← дельта + период сравнения
│  ▔▔▔▁▁▔▔▔▁▔▔▔▔▔    │  ← мини-спарклайн
└─────────────────────┘
```

**Правило 5 секунд**: пользователь должен оценить картину за 5 секунд = макс 5-10 KPI. Самые важные — в левом верхнем углу.

## 2. ВОРОНКИ (FUNNEL)

### Классическая воронка

```
█████████████████████████  Лиды        1343  (100%)
████████████████████       Показы       900   (67%)
█████████████              Сделки       425   (31.6%)
████████                   Регистрация  283   (21%)
████                       Закрытие     156   (11.6%)
```

### Горизонтальная воронка (экономит пространство)

```css
.funnel-stage {
  height: 48px;
  border-radius: 8px;
  display: flex;
  align-items: center;
  justify-content: space-between;
  padding: 0 16px;
  margin-bottom: 4px;
  transition: width 0.6s ease;
}
```

### Конверсии между этапами

```
Лиды 1343 ──[67%]──> Показы 900 ──[47%]──> Сделки 425
```

Цвет конверсии: зелёный >= целевого, красный < целевого.

Для Битрикс24 Estate Invest (4 воронки) — показывать каждую отдельно или в виде табов.

**Библиотеки:** ApexCharts (type: 'bar' горизонтальный), D3.js (Sankey), или чистый CSS.

## 4. ДАШБОРДЫ ДЛЯ НЕДВИЖИМОСТИ

### Ключевые метрики

- **Стоимость портфеля** (суммарная) — крупная KPI
- **ROI** — процент с дельтой
- **Occupancy Rate** — заполняемость
- **Cash Flow** — приход минус расход
- **Количество объектов** — с разбивкой по статусам

### Карточка объекта

```
┌─────────────────────────────────┐
│ [фото]  ул. Ленина, 15          │
│          Калининград             │
│          ● Продано               │
│  Покупка: 1.8M  |  Продажа: 3.2M │
│  Прибыль: +1.4M (+77%)         │
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓░░░ 77% ROI    │
└─────────────────────────────────┘
```

### Для Estate Invest конкретно

- **Сводная панель**: покупка 196M, продажа 388M, прибыль 37M
- **Карта объектов** (Leaflet.js) — точки с попапами
- **Таблица объектов** — сортировка по ROI, городу, статусу

## 6. СОВРЕМЕННЫЕ ТАБЛИЦЫ

### Базовые принципы

- **Чередование строк**: `background: rgba(255,255,255,0.03)` на нечётных
- **Числа — вправо**, текст — влево
- **Фиксированный заголовок**: `position: sticky; top: 0`
- **Hover**: `background: rgba(59,130,246,0.1)`
- **Тонкие бордеры**: только `border-bottom: 1px solid #1e293b`

### CSS для финансовой таблицы

```css
.data-table { width: 100%; border-collapse: collapse; font-size: 0.875rem; }
.data-table th {
  text-align: left; padding: 12px 16px;
  font-size: 0.75rem; text-transform: uppercase; letter-spacing: 0.05em;
  color: #64748b; border-bottom: 2px solid #334155;
  position: sticky; top: 0; background: #0f172a;
}
.data-table td { padding: 12px 16px; border-bottom: 1px solid #1e293b; color: #e2e8f0; }
.data-table tr:hover { background: rgba(59,130,246,0.05); }
.data-table td.number { text-align: right; font-variant-numeric: tabular-nums; }
.data-table td.positive { color: #22c55e; }
.data-table td.negative { color: #ef4444; }
```

### Мобильная адаптация

- Горизонтальный скролл (`overflow-x: auto`)
- Или card-view (каждая строка = карточка)
- Скрывать второстепенные колонки через media query

## 8. УВЕДОМЛЕНИЯ / АЛЕРТЫ

### 4 уровня важности

1. **Критический** (красный) — немедленные действия
2. **Предупреждение** (янтарный) — дедлайн приближается
3. **Информация** (синий) — обновление
4. **Успех** (зелёный) — подтверждение

### CSS

```css
.alert {
  display: flex; align-items: flex-start; gap: 12px;
  padding: 16px; border-radius: 12px; border-left: 4px solid;
  margin-bottom: 8px;
}
.alert--critical { background: rgba(239,68,68,0.08);  border-color: #ef4444; }
.alert--warning  { background: rgba(245,158,11,0.08); border-color: #f59e0b; }
.alert--info     { background: rgba(59,130,246,0.08);  border-color: #3b82f6; }
.alert--success  { background: rgba(34,197,94,0.08);   border-color: #22c55e; }
```

Расположение: панель справа или лента вверху, критические — первыми.

## 9. СРАВНЕНИЯ И ДЕЛЬТЫ

### Паттерн дельты

```
196M ₽  ▲ +23.5%  vs Q3
```

- `▲` / `↑` — рост (зелёный)
- `▼` / `↓` — падение (красный)
- `→` / `—` — без изменений (серый)

```css
.delta { display: inline-flex; align-items: center; gap: 4px; font-size: 0.875rem; font-weight: 500; }
.delta--up   { color: #22c55e; }
.delta--down { color: #ef4444; }
.delta--flat { color: #64748b; }
```

### Before/After

```
┌──────────────┬──────────────┐
│  Q3 2025     │  Q4 2025     │
│  156M ₽      │  196M ₽      │
│              │  ▲ +25.6%    │
└──────────────┴──────────────┘
```
