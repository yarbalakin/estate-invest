## Детальный разбор каждого инструмента

### 1. Yandex DataLens (РЕКОМЕНДУЕТСЯ)

- **Цена**: бесплатно (Community), Business — 990 руб/мес за пользователя
- **Коннекторы**: Google Sheets (нативный), ClickHouse, PostgreSQL, MySQL
- **Битрикс24**: через Albato или REST API → PostgreSQL → DataLens
- **Плюсы**: бесплатный, российский, open-source (Docker self-host), кэш Sheets обновляется ~5 мин
- **Минусы**: меньше коннекторов чем Looker, ограниченная кастомизация
- **Docker self-host**: https://github.com/datalens-tech/datalens
- **Быстрый старт**: https://vc.ru/id1240117/458255

### 2. Google Looker Studio

- **Цена**: бесплатно без ограничений
- **Коннекторы**: Google Sheets (лучшая интеграция), 1270+ коннекторов
- **Битрикс24**: через Coupler.io или Supermetrics (платные)
- **Плюсы**: нулевой порог входа, отличная интеграция с Google-экосистемой
- **Минусы**: серверы в США, нет real-time, ограниченная аналитика
- **Подходит для**: быстрый старт, всё уже в Google Sheets

### 3. Grafana OSS

- **Цена**: бесплатно self-hosted, бесплатный Cloud (до 10K метрик)
- **Docker**: `docker run -d -p 3000:3000 grafana/grafana-oss`
- **Плюсы**: авто-обновление панелей (30 сек), 80+ data-source плагинов, алерты из коробки
- **Минусы**: ориентация на time-series, не бизнес-отчёты, кривая обучения
- **Подходит для**: мониторинг KPI в реальном времени, алерты

### 4. Metabase

- **Цена**: Open Source — бесплатно, Pro — $85/мес (5 users)
- **Docker**: `docker run -d -p 3000:3000 metabase/metabase`
- **Плюсы**: GUI для запросов без SQL ("Ask a Question"), embedded analytics, 90К+ компаний
- **Минусы**: нет прямого REST API подключения (нужна БД), нет real-time
- **Google Sheets**: только в Metabase Cloud (не self-hosted)
- **Подходит для**: бизнес-аналитика, самообслуживание для команды

### 5. Streamlit (Python)

- **Цена**: бесплатно, Streamlit Cloud — до 3 приложений
- **Плюсы**: полная кастомизация, Python/Pandas, ML-интеграция, бесплатный хостинг
- **Минусы**: нет real-time (рефреш страницы), 1 GB RAM Cloud, app засыпает, public repo
- **Уже есть**: наработки в `/Users/yaroslavbalakin/Desktop/Projects/Эстейт/` (data_loader.py, analytics.py, app.py)
- **Подходит для**: ML-дашборды, прототипирование, полный контроль

### 6. Retool

- **Цена**: бесплатно (5 users) / $10/user/мес
- **Плюсы**: drag-and-drop, прямое подключение к REST API, быстрый старт
- **Минусы**: нет self-host, vendor lock-in, дорого при масштабе
- **Подходит для**: быстрые internal tools, если есть бюджет

### 7. Next.js + Tremor/Shadcn

- **Цена**: бесплатно (Vercel деплой)
- **Шаблоны**: TailAdmin, next-shadcn-dashboard-starter, Devias Kit
- **Плюсы**: полный контроль UI/UX, WebSocket real-time, 35+ компонентов Tremor
- **Минусы**: TypeScript/React навыки, 1-2 недели разработки
- **Подходит для**: кастомные решения, SaaS-продукты
