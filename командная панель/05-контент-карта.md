## 5. Алерты

### Критические (красные)
- Анкета не обработана > 48ч (Битрикс)
- Объект просрочен по сроку реализации (Sheets)
- Расхождение сумм сбора (Sheets)
- Инвестор запросил возврат (Sheets)

### Важные (жёлтые)
- Сбор заполнен > 90%
- Объект без активности > 30 дней
- Торги через 3 дня
- Крупный инвестор > 500K

### Информационные (серые)
- Еженедельная сводка по воронке
- Новые регистрации за неделю

## 6. Drill-down при клике на KPI

| KPI | Открывает |
|-----|----------|
| Капитал 196M | Финансы → разбивка по объектам (bar) + по валютам (donut) |
| Продажа 388M | Объекты → таблица с покупка/продажа/ROI |
| Прибыль 37M | Финансы → Выплаты по месяцам (stacked bar) |
| Инвесторы 900/2000 | Инвесторы → активные vs неактивные, рост по месяцам |
| Объектов N/118 | Объекты → Канбан |
| Конверсия 37% | Воронки CRM → Регистрация (funnel) |
| Цель 24/год | Торги + Объекты → timeline + прогноз |

## 7. Специфичные виджеты для недвижимости

1. **Калькулятор ROI объекта** — внутри карточки объекта
2. **Progress Bar сбора средств** — для объектов со стадией "сбор"
3. **Timeline объекта** — от выкупа до продажи с маркерами событий
4. **Heatmap по городам** — Пермь ~60%, Калининград ~20%, Самара ~10%
5. **Долговой трекер** — обязательства vs ликвидная стоимость
6. **Виджет "Следующие выплаты"** — кому, сколько, когда
