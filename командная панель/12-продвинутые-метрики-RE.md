# Продвинутые метрики для инвестиционной компании Estate Invest

---

## 1. Инвестиционные метрики

### IRR (Internal Rate of Return)
```
0 = Sum[ CF_t / (1 + IRR)^t ]  для t = 0...n
```
Учитывает время: объект с быстрым циклом (3 мес) даёт выше IRR чем медленный (9 мес) при одинаковой прибыли.

**Бенчмарки**: Core RE 6-10%, Value-add (Estate Invest) **12-18%**, Opportunistic 18-25%+

### MOIC (Multiple on Invested Capital)
```
MOIC = Общие возвраты / Общий вложенный капитал
```
**Estate Invest**: ~196M покупка, ~388M продажа → MOIC ~1.98x (отличный для value-add, бенчмарк 1.7-2.0x)

### TVPI (Total Value to Paid-In)
```
TVPI = (Распределения + Остаточная стоимость) / Вызванный капитал
TVPI = DPI + RVPI
```
Полная картина: выплаченное + ещё "в работе". Бенчмарк > 1.5x.

### DPI (Distributions to Paid-In)
```
DPI = Распределения инвесторам / Вызванный капитал
```
"Реальные деньги в кармане". DPI > 1.0x = инвестор вернул капитал. Бенчмарк 1.6x = 60% прибыли.

### Equity Multiple
```
Equity Multiple = Все распределения / Собственный капитал
```
Бенчмарк value-add: 1.7x-2.0x+

### J-Curve
Типичный паттерн: первые 1-3 года отрицательные (покупка, ремонт), потом рост. У Estate Invest J-curve короче (месяцы, не годы). Визуализация помогает управлять ожиданиями инвесторов.

## 2. Здоровье портфеля

### HHI (Индекс Херфиндаля-Хиршмана) — концентрация
```
HHI = Sum(доля_i^2)  где доля в %
```
**Шкала**: < 1500 (хорошая диверсификация), 1500-2500 (умеренная), > 2500 (РИСК)

**Считать по**: городам, типам объектов, инвесторам, ценовым сегментам.

**Пример Estate Invest** (60% Пермь, 20% Калининград, 10% Самара, 10% другие):
```
HHI = 60² + 20² + 10² + 10² = 3600 + 400 + 100 + 100 = 4200 (высокая концентрация!)
```

### Vintage Diversification
Распределение покупок по кварталам/годам. Если всё куплено в одном периоде = зависимость от условий одного рынка.

### LTV / LTC
```
LTV = Займы / Рыночная стоимость
LTC = Займы / Стоимость приобретения
```
Conservative: LTV < 50%, Moderate: 50-65%, Aggressive: > 65%

### Liquidity Ratio
```
Liquidity = (Свободные средства + Готовые к продаже) / Обязательства
```

## 3. Операционная эффективность

### Hold Period (Time-to-Close)
```
Hold Period = Дата продажи - Дата покупки (дни)
```
**Бенчмарк flip**: 2025 средний 183 дня. Каждые +30 дней сверх 180 = минус 1-1.5% от ROI.

**Разбивка по фазам**:
```
Acquisition Time = покупка - аукцион
Renovation Time = завершение ремонта - покупка
Sale Time = продажа - выставление
Total = Acquisition + Renovation + Sale
```

### Renovation Efficiency
```
Cost Variance = (Факт - План) / План * 100%
Schedule Variance = (Факт дней - План дней) / План дней * 100%
Renovation ROI = (Прирост стоимости - Стоимость ремонта) / Стоимость ремонта
```
**Бенчмарк**: Cost Variance < 5% (хорошо), Renovation ROI > 200% (отлично)

### Pipeline Velocity
```
Velocity = (Сделки в воронке * Средняя сумма * Win Rate) / Средняя длина цикла (дни)
```
Показывает ожидаемый ежедневный revenue.

### Staff Productivity
```
Объектов/сотрудник = 118 / 26 ≈ 4.5
Revenue/сотрудник = Выручка / Кол-во сотрудников
```

## 4. Investor Relations

### NAV (Net Asset Value)
```
NAV = Справедливая стоимость объектов - Обязательства - Запланированный CAPEX
NAV per investor = NAV * Доля инвестора / Общий капитал
```
Ежеквартально минимум. Каждый из 118 объектов должен иметь текущую оценку.

### Distribution Yield
```
Distribution Yield = Годовые распределения / Стоимость доли * 100%
```
Бенчмарк value-add: 6-10%.

### NPS инвесторов
```
NPS = % Промоутеров (9-10) - % Детракторов (0-6)
```
**Шкала**: < 0 критично, 0-20 нормально, 20-50 хорошо, 50+ отлично. RE средний = 30.

### Communication Metrics
- Response Time — среднее время ответа
- Report Frequency — отчёты за квартал
- Engagement Rate — % открывших отчёт
- Query Resolution Time — время решения вопроса

## 5. Предиктивные (опережающие) индикаторы

| Метрика | Формула | Что показывает |
|---------|---------|----------------|
| Pipeline-to-Close | Объекты в воронке / Закрытые | Рост = снижение качества pipeline |
| Auction Win Rate | Выиграно / Участий | Падение = рост конкуренции |
| Sales-to-Listings | Продано / Выставлено | > 50% рынок продавца |
| Inquiry-to-Invest | Новые инвесторы / Лиды | Эффективность привлечения |

### Macro Leading Indicators
- Объём банковских аукционов (рост = возможности)
- Ключевая ставка ЦБ (рост = дешевле покупать, дороже для покупателей)
- Объём ипотеки (падение = меньше покупателей)

## 6. Composite Health Score (Единый индекс здоровья)

### Методология

**Шаг 1: Компоненты и веса**
```
1. MOIC (финансы)                    25%
2. Hold Period (операции)             20%
3. Win Rate + Velocity (pipeline)     15%
4. NPS (инвесторы)                    15%
5. HHI inverse (риск)                10%
6. Рост инвесторов MoM               10%
7. DPI (ликвидность)                   5%
```

**Шаг 2: Нормализация на 0-100**
```
Score = (Actual - Worst) / (Best - Worst) * 100
```

**Шаг 3: Взвешенная сумма**
```
Health Score = Sum(Normalized_i * Weight_i)
```

**Шаг 4: Интерпретация**
```
90-100: Отлично (зелёная)
70-89:  Хорошо (зелёная-жёлтая)
50-69:  Внимание (жёлтая)
30-49:  Проблема (оранжевая)
0-29:   Кризис (красная)
```

### Пример расчёта для Estate Invest
```
MOIC 1.98x       → norm 65.3  × 0.25 = 16.3
Hold 150 дн      → norm 78.2  × 0.20 = 15.6
Win Rate 37%     → norm 54.0  × 0.15 =  8.1
NPS 45           → norm 65.0  × 0.15 =  9.8
HHI inv 4450     → norm 61.7  × 0.10 =  6.2
Growth 5% MoM    → norm 33.3  × 0.10 =  3.3
DPI 1.2x         → norm 46.7  × 0.05 =  2.3
─────────────────────────────────────────
HEALTH SCORE = 61.6 / 100 (жёлтая — "Внимание")
```

Тренд Health Score важнее абсолютного значения. Пересчитывать ежемесячно.

## 7. Когортный анализ инвесторов

### Когорта = группа по дате первой инвестиции

**Ключевые метрики по когортам**:

**Reinvestment Rate**
```
= Повторно вложившие / Всего в когорте * 100%
```
Бенчмарк: > 40% = отличная лояльность.

**Retention по когортам** (SaaS-стиль):
```
          +3 мес  +6 мес  +12 мес  +18 мес
Q1-2023:   95%     88%     72%      65%
Q2-2023:   92%     85%     70%      62%
Q3-2023:   97%     90%     75%       —
```
"Retention" = % инвесторов, чьи деньги ещё работают.

**Average Investment Growth**
```
= Средняя 2-я инвестиция / Средняя 1-я
```
Рост > 1.0 = доверие растёт.

**LTV инвестора**
```
= Средняя инвестиция × Кол-во инвестиций × Средний срок × Маржа
```

**Ключевой инсайт**: Если ранние когорты = reinvestment 60%, поздние = 30%, значит что-то изменилось в продукте или привлечении.
