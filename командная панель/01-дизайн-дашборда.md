# Дизайн executive-дашборда — Полное исследование

> Собрано: 25.02.2026. Источники: 30+ статей, Dribbble, Behance, документация.

---

## 1. LAYOUT-ПАТТЕРНЫ

### Главная схема: Sidebar + Top Rail + Content Grid

Доминирующий паттерн 2025-2026 — **сворачиваемый сайдбар слева** (как у Vercel, Linear, Stripe) + горизонтальная полоса KPI сверху + основной контент в grid-сетке.

**Конкретная структура:**
- **Sidebar**: 240-280px в развёрнутом виде, 64-72px в свёрнутом (только иконки)
- **Top Rail**: высота 56-64px, содержит breadcrumbs / фильтры / период
- **KPI-полоса**: первый ряд под top rail, 3-5 карточек в линию
- **Content area**: CSS Grid `grid-template-columns: repeat(auto-fit, minmax(320px, 1fr))` с gap 16-24px

### Bento Grid Layout — тренд 2025-2026 (вдохновлён Apple)

Карточки разного размера в единой сетке:
```css
.dashboard-grid {
    display: grid;
    grid-template-columns: repeat(12, 1fr);
    gap: 16px;
    padding: 24px;
}
.card-small  { grid-column: span 3; }  /* 1/4 ширины */
.card-medium { grid-column: span 6; }  /* 1/2 ширины */
.card-large  { grid-column: span 12; } /* полная ширина */
```

### Принцип "перевёрнутая пирамида"
- Верх — ключевые KPI (самое важное)
- Середина — тренды и графики
- Низ — детальные таблицы и breakdown

### Spacing System (кратно 4px или 8px)
- Внутренний padding карточек: 20-24px
- Gap между карточками: 16-24px
- Отступ от sidebar: 24-32px
- Между секциями: 32-48px

## 2. ЦВЕТОВЫЕ СХЕМЫ

### Dark Mode (основной тренд 2025-2026)

Тёмная тема — **стандарт** для CEO-дашбордов. Уменьшает усталость глаз, выглядит premium, яркие данные лучше выделяются.

**Рекомендуемая dark-палитра:**

| Элемент | Hex |
|---------|-----|
| Фон страницы | `#0a0a0f` или `#111118` |
| Фон sidebar | `#141420` |
| Фон карточки | `#1a1a2e` или `#181824` |
| Фон карточки hover | `#222238` |
| Основной текст | `#e4e4e7` или `#f0f0f3` |
| Вторичный текст | `#71717a` или `#6c7293` |
| Бордер карточек | `rgba(255,255,255,0.06)` |
| Бордер hover | `rgba(255,255,255,0.12)` |

**Акцентные цвета для метрик:**

| Назначение | Hex |
|-----------|-----|
| Рост / Прибыль / Успех | `#22c55e` |
| Падение / Убыток / Ошибка | `#ef4444` |
| Предупреждение | `#eab308` |
| Primary accent | `#8170ed` (фиолетовый) |
| Secondary accent | `#3b86d1` (синий) |

**Палитра для графиков (6 цветов):**
```
#6366f1 (индиго)
#8b5cf6 (фиолетовый)
#06b6d4 (циан)
#22c55e (зелёный)
#eab308 (жёлтый)
#ef4444 (красный)
```

### Light Mode (альтернатива)

## 3. ТИПОГРАФИКА

### Основные шрифты

**Заголовки и UI-текст:**
- **Inter** — стандарт индустрии. Google Fonts, бесплатный. Используется в Linear, многих SaaS
- **Geist Sans** — шрифт Vercel. Геометрический sans-serif, 9 весов
- **DM Sans** — мягче Inter, более дружелюбный

**Числа и данные:**
- **Geist Mono** / **JetBrains Mono** — для моноширинных данных
- Для KPI лучше **tabular numerals** в пропорциональном шрифте:
```css
.kpi-value { font-variant-numeric: tabular-nums; }
```

### Размеры текста

```css
:root {
    --font-sans: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
    --font-mono: 'JetBrains Mono', 'Fira Code', monospace;

    --text-xs:   11px;   /* мелкие лейблы, timestamp */
    --text-sm:   13px;   /* таблицы, вторичный текст */
    --text-base: 14px;   /* основной текст дашборда */
    --text-lg:   16px;   /* подзаголовки карточек */
    --text-xl:   20px;   /* заголовки секций */
    --text-2xl:  24px;   /* KPI-числа (средние) */
    --text-3xl:  32px;   /* KPI-числа (крупные) */
    --text-4xl:  48px;   /* hero-метрика */

    --leading-tight:  1.2;  /* заголовки, KPI */
    --leading-normal: 1.4;  /* таблицы */
    --leading-relaxed: 1.6; /* body text */
}
```

**Ключевое правило:** текст в дашборде редко превышает 14px. Таблицы лучше на **13px, line-height 1.4**.

## 4. ВИЗУАЛИЗАЦИЯ ДАННЫХ

### Как представлять разные типы:

| Тип данных | Как показать |
|-----------|-------------|
| **Деньги/Выручка** | KPI-карточка 32-48px + sparkline + % изменения |
| **Проценты** | Donut/Ring chart (число в центре) или progress bar |
| **Тренды роста** | Area chart с градиентной заливкой |
| **Кол-во сотрудников/объектов** | KPI-карточка + stacked bar chart |
| **Таймлайны** | Horizontal timeline с milestones |
| **Воронки/Стадии** | Horizontal funnel chart |

**Формат чисел:** `12.5 млн` или `$12.5M` (сокращать до одного знака после запятой)

**Progressive Disclosure:** сначала обзор, детали — по клику.

## 5. KPI-КАРТОЧКИ (Анатомия)

### Структура идеальной карточки (5 элементов):

```
+----------------------------------+
|  Период (мес/год)         ...    |  <- text-xs, secondary
|                                   |
|  Выручка                         |  <- text-sm, weight-medium
|  12.5 млн                        |  <- text-3xl (32px), bold
|  +23.5% ▲                        |  <- text-sm, green/red
|                                   |
|  ~~~~ sparkline ~~~~             |  <- 40-60px высота, без осей
+----------------------------------+
```

### CSS:

```css
.kpi-card {
    background: rgba(255, 255, 255, 0.04);
    border: 1px solid rgba(255, 255, 255, 0.06);
    border-radius: 16px;
    padding: 20px 24px;
    display: flex;
    flex-direction: column;
    gap: 8px;
    transition: all 0.2s ease;
}
.kpi-card:hover {
    background: rgba(255, 255, 255, 0.08);
    border-color: rgba(255, 255, 255, 0.12);
    transform: translateY(-2px);
    box-shadow: 0 8px 24px rgba(0, 0, 0, 0.2);
}
.kpi-label { font-size: 13px; color: #71717a; font-weight: 500; text-transform: uppercase; letter-spacing: 0.05em; }
.kpi-value { font-size: 32px; font-weight: 700; color: #f0f0f3; font-variant-numeric: tabular-nums; line-height: 1.2; }
.kpi-trend { font-size: 13px; font-weight: 600; display: flex; align-items: center; gap: 4px; }
.kpi-trend.positive { color: #22c55e; }
.kpi-trend.negative { color: #ef4444; }
```

### Правила:
- Выравнивание — всегда left-align
- Не более **5-7 KPI** на экране
- Каждое число должно иметь **контекст** (vs прошлый период, vs план)

## 6. ГРАФИКИ — Лучшие типы

| Тип | Применение |
|-----|-----------|
| **Sparkline** | Внутри KPI-карточки, тренд без деталей |
| **Area Chart** (с градиентом) | Выручка, трафик — объём во времени |
| **Line Chart** | Сравнение 2-4 метрик |
| **Bar Chart** (вертикальный) | Помесячная выручка, сравнение периодов |
| **Horizontal Bar** | Топ-5, рейтинг |
| **Donut Chart** | Доли/проценты, структура выручки |
| **Stacked Bar** | Состав целого по месяцам |
| **Funnel** | Конверсия, стадии сделок |
| **Heatmap** | Активность по времени |

**Что НЕ использовать:** 3D-графики, полные pie charts, gauge charts.

**Библиотеки:**
- **ApexCharts** — лучший баланс красоты и лёгкости, встроенные sparklines
- **Chart.js** — классика, легковесный
- **Lightweight Charts (TradingView)** — для финансовых графиков

## 8. МИКРО-ВЗАИМОДЕЙСТВИЯ

### Hover карточек:
```css
.card {
    transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1);
}
.card:hover {
    transform: translateY(-2px);
    box-shadow: 0 8px 24px rgba(0, 0, 0, 0.15);
    border-color: rgba(255, 255, 255, 0.15);
}
```

### Анимация появления:
```css
@keyframes fadeInUp {
    from { opacity: 0; transform: translateY(12px); }
    to { opacity: 1; transform: translateY(0); }
}
.card { animation: fadeInUp 0.3s ease forwards; }
.card:nth-child(1) { animation-delay: 0ms; }
.card:nth-child(2) { animation-delay: 50ms; }
.card:nth-child(3) { animation-delay: 100ms; }
```

### Ключевые принципы:
- **200ms** — оптимальная длительность переходов
- `cubic-bezier(0.4, 0, 0.2, 1)` — стандартная easing Material Design
- Только `transform` и `opacity` для 60fps

## 10. CSS-ТЕХНИКИ

### Glassmorphism:
```css
.glass-card {
    background: rgba(255, 255, 255, 0.04);
    backdrop-filter: blur(12px) saturate(180%);
    border: 1px solid rgba(255, 255, 255, 0.08);
    border-radius: 16px;
    box-shadow: 0 8px 32px rgba(0, 0, 0, 0.25);
}
```

### Градиентный фон:
```css
body {
    background: #0a0a0f;
    background-image:
        radial-gradient(ellipse at 20% 50%, rgba(99, 102, 241, 0.08) 0%, transparent 50%),
        radial-gradient(ellipse at 80% 20%, rgba(139, 92, 246, 0.06) 0%, transparent 50%),
        radial-gradient(ellipse at 60% 80%, rgba(6, 182, 212, 0.04) 0%, transparent 50%);
}
```

### Современные border-radius:
- Карточки: `16px`
- Кнопки: `8-12px`
- Pill-бейджи: `9999px`
- Аватарки: `50%`

### Gradient border:
```css
.gradient-border::before {
    content: '';
    position: absolute;
    inset: 0;
    border-radius: 16px;
    padding: 1px;
    background: linear-gradient(135deg, rgba(99,102,241,0.5), rgba(139,92,246,0.2), transparent);
    -webkit-mask: linear-gradient(#fff 0 0) content-box, linear-gradient(#fff 0 0);
    -webkit-mask-composite: xor;
    mask-composite: exclude;
}
```
