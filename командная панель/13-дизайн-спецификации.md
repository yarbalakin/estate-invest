## 2. Component Sizing

### KPI-карточка (анатомия)
```
1. Label (название)     12-14px, muted color
2. Headline Value        36-48px bold, primary
3. Delta (%, изменение)  14px, semantic color
4. Sparkline/иконка      32-48px height
```

### Размеры
```
Карточка min-height:     120-160px
Padding внутренний:      24px (p-6)
Gap между карточками:    16px (gap-4)
Border-radius:           12px (rounded-xl)
Hero number:             36-48px (text-4xl / text-5xl)
Dashboard-level hero:    48-72px (text-6xl / text-7xl)
```

### Графики
```
Основной график:         min-height 300-400px
Вторичный:               min-height 200-300px
Sparkline inline:        32-48px height
Sparkline в карточке:    48-64px height
```

## 6. Chart Patterns

### Выбор типа
```
Тренд (1 серия)     → Line / Area
Тренд (2-5 серий)   → Multi-line
Категории           → Bar (horizontal для длинных лейблов)
Доли                → Donut / Stacked bar
KPI + тренд         → Sparkline (line)
Показатель vs цель  → Progress bar / Gauge
```

### Sparkline спецификации
```
Inline:   80-120px wide, 32-40px high, stroke 1.5-2px
В карточке: 100% wide, 48-64px high, stroke 2px
Без осей, без лейблов
Точка на последнем значении, fill area opacity 0.1
```

### Annotations
```
Threshold line:    dashed, 1px, amber-500
Target line:       dashed, 2px, slate-400
Period highlight:  background fill, opacity 0.05
Tooltip:           bg-popover, rounded-md, shadow-md, p-3
```

## 8. Responsive Breakpoints

```
Default (0+):   1 column, cards stack
sm (640px+):    2 columns
md (768px+):    sidebar visible, 2 col charts
lg (1024px+):   4 col KPI, 7-col chart split
xl (1280px+):   max-width container
```

### Mobile priority
1. KPI-карточки остаются (1 колонка)
2. Основной график (100% ширины)
3. Вторичные → ниже или за табами
4. Sidebar → drawer
5. Таблица → горизонтальный скролл
