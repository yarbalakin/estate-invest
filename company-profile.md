## 2. Бизнес-модель

### Цикл работы с объектом

```
Поиск → Выкуп → Создание ценности → Продажа → Распределение прибыли
```

| Этап | Что происходит |
|------|----------------|
| **Поиск и анализ** | Мониторинг торгов, анализ потенциала, финансовая модель |
| **Выкуп** | Участие в торгах, выкуп ниже рынка, оформление собственности |
| **Создание ценности** | Ремонт, раздел помещений, межевание, техдокументы |
| **Реализация** | Размещение на Авито, показы, переговоры, закрытие сделок |
| **Распределение** | Расчёт доходности, начисление прибыли через ЛК, выплаты |

### Финансовая модель

- Минимальный вход: **от 1 000 руб.** в конкретный объект (ИП)
- Распределение прибыли: **50/50** (компания / инвесторы, пропорционально вкладу)
- Юридическая основа: рамочный инвестиционный договор (шаблон от 21.08.2023)
- Подписание: электронная подпись через **SignEasy**
- Арбитраж: третейский судья (ФЗ-102)
- Налоги: инвестор платит НДФЛ 13% самостоятельно

### Виды операций

| Операция | Описание |
|----------|----------|
| вложение / довложение | Первоначальный и дополнительный взнос |
| конвертация | Перевод между валютами |
| возврат | Возврат средств инвестору |
| бонус / реферальный бонус | Бонусные начисления, 10% за реферала |
| командные | Командные выплаты |
| перевод минус/плюс | Внутренние переводы между объектами |
| выплата тела / выплата % | Возврат основной суммы + доля прибыли |
