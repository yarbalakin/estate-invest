# Estate Аналитика — Сводка исследования

**Дата**: 2026-03-01
**Цель**: Инструмент аналитики недвижимости для Estate Invest + инвесторов
**MVP-регион**: Пермь

---

## Что это за проект

Система, которая:
1. Скринит все лоты на торговых площадках (torgi.gov.ru, банкротные торги)
2. Оценивает рыночную стоимость каждого лота (квартиры, земля, коммерция)
3. Показывает дисконт к рынку и рекомендацию "купить/пропустить"

Целевая аудитория: Estate Invest + независимые инвесторы в недвижимость.

---

## Ключевые выводы из исследования

### 1. Рынок — зрелые агрегаторы, слабая аналитика

- **10+ агрегаторов** торгов в РФ (TBankrot, Bankrot-Spy, EI.ru, Rutorgi, Finderlot и др.)
- Но **глубокую AI-аналитику с оценкой не делает никто**
- TBankrot — ближайший конкурент (90 000+ лотов, тарифы 1 500-5 800 руб/мес)
- **Telegram-first продукта с аналитикой не существует** — белое пятно
- В США аналог: PropertyRadar ($49-79/мес, 150M+ объектов)

### 2. Мировые лидеры — как они устроены

| Платформа | Модель | Точность | Данные |
|-----------|--------|----------|--------|
| Zillow Zestimate | Neural network | 1.83% on-market / 7.01% off-market | 100M+ домов, MLS, налоги, фото |
| Redfin Estimate | ML + MLS | 1.98% / 7.72% | MLS (прямой доступ как брокер) |
| HouseCanary | ML + CV | 2.7% post-list / 7.5% pre-list | 114M+ объектов, фото, comparables |
| CoreLogic | ML (unified) | не раскрывают | 99.9%+ покрытие США, 50 лет истории |
| Домклик (Сбер) | LightGBM | не раскрывают | ипотечные сделки Сбера (уникально) |
| ЦИАН | ML | не раскрывают | 200 параметров, 7 лет архива |

**Ключевой инсайт**: LightGBM/XGBoost доминируют в production AVM. Нейросети — только у Zillow.

### 3. Источники данных для MVP (Пермь)

| Источник | Юр. риск | API | Данные | Приоритет |
|----------|----------|-----|--------|-----------|
| **torgi.gov.ru** | Минимальный (Open Data!) | **REST API (JSON!)** | Лоты торгов, цены, описания | **P0** |
| **ЕФРСБ** | Низкий | POST API + CAPTCHA | Банкротства, дела, лоты | **P0** |
| **ads-api.ru** | Нет (легальный агрегатор) | REST API | ЦИАН+Авито+Домклик | **P1 (рекомендуется)** |
| **Avito** | Высокий | Нет | Рыночные цены, описания | P2 (fallback) |
| **ЦИАН** | Высокий | Внутренний JSON | Цены + координаты + атрибуты | P2 (fallback) |
| **PKK Росреестр** | Средний | Неофициальный | Кадастр, площадь, стоимость | **P1** |
| **DaData** | Нет | Официальный (10K/день free) | Геокодинг, адреса, FIAS GUID | **P1** |

**Новое**: torgi.gov.ru имеет полноценный **JSON REST API** (не только XML). Подробности в TECHNICAL_DEEP_DIVE.md.
**Новое**: **ads-api.ru** — легальный агрегатор ЦИАН+Авито+Домклик, 14 дней бесплатно, ~3000 руб/мес. Снимает юр. риски парсинга.
**Avito/ЦИАН — высокий риск**: прецеденты ФАС (Avito vs ЦИАН, VK vs Double). Собственные парсеры — как fallback.

### 4. Техническая архитектура

**n8n НЕ подходит как основной инструмент** (5000+ листингов, нет Playwright, нет PostGIS, нет ML).

**Рекомендуемый стек**:
```
Сбор данных:  Python (Scrapy + Playwright) → cron
Хранилище:    PostgreSQL + PostGIS (пространственные запросы критичны)
Оценка:       Python (pandas, scikit-learn, XGBoost)
Интерфейс:    Telegram бот (MVP) → Web dashboard (v2)
Оркестрация:  n8n (расписание, уведомления, Telegram)
Геокодинг:    DaData API (10K бесплатных запросов/день)
```

**Гибрид — правильный путь**: n8n как оркестратор, Python-бэкенд для тяжёлой работы.

### 5. Алгоритм оценки

**Фаза 1 (MVP)**: Rule-based comparable sales + price/sqm по районам
- Коэффициенты корректировки (этаж, ремонт, тип дома, площадь)
- Не требует ML, работает с малым количеством данных

**Фаза 2**: XGBoost/LightGBM (нужно 2000+ объектов с признаками)
- Feature engineering: 20+ признаков (площадь, этаж, расстояние до центра, тип дома...)
- SHAP для объяснения оценки

**Фаза 3**: Ensemble + computer vision (фото для оценки состояния)

### 6. Дисконты на торгах

| Этап торгов | Дисконт к рынку |
|-------------|----------------|
| Первичные (аукцион на повышение) | 0% (может продаться дороже) |
| Повторные (-10% от начальной) | ~10% |
| Публичное предложение (начало) | 10-20% |
| Публичное предложение (середина) | 30-50% |
| Публичное предложение (глубокое снижение) | 50-75% |
| **Среднее по рынку** | **20-60%** |

### 7. Монетизация

**Фаза 1 (0-6 мес)**: бесплатный бот → набор аудитории
**Фаза 2 (6-12 мес)**: freemium — бесплатные уведомления, платная аналитика (1 500 руб/мес)
**Фаза 3 (12+ мес)**: отчёты по лотам (500-2000 руб), API, агентская модель

### 8. Конкурентные преимущества (дифференциация)

1. **AI-оценка рыночной стоимости** — никто в РФ не делает хорошо
2. **Telegram-first UX** — все конкуренты web-first
3. **Фокус на недвижимость** — как PropertyRadar начинал с foreclosures
4. **Интеграция с кадастром** — автообогащение лотов данными Росреестра
5. **Компетенция Estate Invest** — реальный опыт 118 объектов, 900+ инвесторов

### 9. Зарубежный опыт — что внедрить (раунд 2)

**UK — золотой стандарт прозрачности**:
- Land Registry: 20M+ реальных цен сделок (бесплатно, с 1995 года)
- UPRN — единый ID для каждого объекта (40M), связывает ВСЕ датасеты
- EPC — физические атрибуты (площадь, комнаты, тип дома) — 30M записей, бесплатно
- Hometrack AVM: 18 из 20 крупнейших ипотечных банков, 50M+ оценок/год, точность 80% в 10%
- PropertyData: browser extension поверх порталов (данные overlay на листингах)
- EIG: 600K+ аукционных лотов в базе, 400 аукционных домов

**Франция — открыли данные в 2019, и всё взлетело**:
- DVF: ВСЕ сделки открыто (CSV, SQL, GeoPackage, API)
- MeilleursAgents: gradient boosting ~1000 деревьев + 12,000 агентств-партнёров

**Германия — нотариусы обязаны отправлять данные о сделках**:
- Gutachterausschuss: юридически обязательная система сбора цен
- BORIS: открытые данные о стоимости земли
- GREIX: 2M+ транзакций с 1960 года (бесплатно)
- ImmoScout24: AVM с точностью ~9.9% vs экспертная оценка

**Австралия — мировой лидер аукционной аналитики**:
- 60%+ объектов в Мельбурне продаются через аукцион
- REIV: clearance rate 3 раза в неделю (preliminary → updated → final)
- CoreLogic AU: 10M+ объектов, 40 лет истории, near-monopoly

**Скандинавия — максимальная прозрачность**:
- Hemnet (Швеция): 90% рынка, финальные цены продаж публичны
- Boliga (Дания): цена предложения vs цена продажи для каждого объекта

**Что внедрить в наш продукт**:
1. Browser extension с overlay данных на ЦИАН/Авито (как PropertyData)
2. Confidence scoring для каждой оценки (как PriceHubble/Hometrack)
3. Clearance rate аукционов (как REIV — preliminary → final)
4. AI-ассистент, объясняющий оценку (как REA Group + OpenAI)
5. SHAP-объяснение каждой оценки (какие факторы повлияли)

**Инструменты для сборки**:
- **OpenAVMKit** — `pip install openavmkit` — FOSS AVM с LightGBM/XGBoost/CatBoost
- **Supabase** — PostgreSQL + PostGIS + API бесплатно до 500MB
- **H3** — гексагональная сетка Uber для price heatmaps
- **Sentence Transformers** — матчинг описаний объектов (дедупликация)
- **CLIP** — бесплатная классификация комнат по фото
- **SHAP** — объяснение оценок ML-модели
