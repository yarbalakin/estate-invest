# RESEARCH_SCORING.md — Исследование алгоритма инвестиционного скоринга

Дата исследования: 2026-04-02
Источники: веб-поиск (3 параллельных агента) — российский + международный опыт.

---

## Текущее состояние (enrich_scoring.py)

Три числовых скора (0–100) + текстовая рекомендация:
- `invest_score` — инвестиционная привлекательность
- `risk_score` — риск (чем ниже — тем лучше)
- `liquidity_score` — ликвидность (скорость продажи)

**Главная проблема:** единая формула для всех типов — квартира, земля, дом, коммерция считаются одинаково. Нужны раздельные функции по типам.

### КВАРТИРЫ

#### Факторы, которых нет в текущем скоринге

**Этаж:**
- 1-й этаж: дисконт 10–20% к рынку (в старом фонде до 20%)
- Последний этаж (старый дом): дисконт 5–10%
- 2–10 этаж с лифтом: эталонная ликвидность

**Материал дома и год постройки:**
- Монолит/кирпич после 2000 г. → +15–20% к рыночной цене vs панель
- Кирпич 1980–2000 → норма
- Хрущёвки и устаревшая панель (1960–1990) → банки часто отказывают покупателю в ипотеке → ликвидность падает на 30–50%
- Деревянные/смешанные дома → банки не принимают как залог

**Состояние ремонта:**
- Хороший ремонт: продаётся быстрее, +5–10% к цене
- Требует косметики: дисконт 5–10%
- Требует капитального ремонта: дисконт 10–15%
- Незаконная перепланировка несущих конструкций: покупатель не получит ипотеку → только за кэш → ликвидность резко падает

**Правовые риски (критические, отсутствуют в коде):**

| Ситуация | Риск | Комментарий |
|----------|------|-------------|
| Прописаны взрослые, согласны выехать | +10 | 1–2 месяца процесс |
| Прописаны взрослые, не согласны | +25 | Суд 4+ мес. |
| Прописаны несовершеннолетние | +30 | Нужна опека |
| Отказник от приватизации | **СТОП** | Нельзя выселить пожизненно (ст. 19 ФЗ-189) |
| МК использован, доли выделены | +20 | Нужно согласие опеки |
| МК использован, доли НЕ выделены | +45 | Сделка может быть оспорена прокуратурой/СФР |
| Единственное жильё без ипотеки | **СТОП** | Не может продаваться в банкротстве |
| Единственное ипотечное жильё (ФЗ-298, 2024) | +30 | Суд может отменить торги |

**Долги:**
- Долги ЖКХ (вода, газ, свет): на нового собственника НЕ переходят
- Долги по взносам на капремонт: **ПЕРЕХОДЯТ** к новому собственнику (ст. 158 ЖК РФ)

#### Рекомендуемые веса для invest_score (квартиры)

| Фактор | Вес |
|--------|-----|
| Дисконт к рыночной цене | 30% |
| Локация (транспорт, инфраструктура, город) | 25% |
| Тип/площадь объекта | 15% |
| Техсостояние дома (год, материал) | 15% |
| Арендный/рентный потенциал | 10% |
| Перспектива роста рынка | 5% |

#### Рекомендуемые веса для liquidity_score (квартиры)

| Фактор | Вес |
|--------|-----|
| Правовая чистота | 20% |
| Площадь (оптимум 28–60 м²) | 20% |
| Этаж | 15% |
| Год постройки и материал | 15% |
| Состояние и ремонт | 15% |
| Размер города/рынок | 15% |

#### Практические цифры
- Дисконт на первых торгах (РФ): 0–15%
- Повторные торги: –10% от начальной цены
- Публичное предложение: средний дисконт –63% от первоначальной цены (данные 2023)
- Целевой дисконт EI: 20–50% к рынку
- Ликвидность 1К до 60 м²: 1–2 мес. экспозиции
- Ликвидность квартир >90 м²: >6 мес. в среднем городе

### ЗЕМЛЯ

#### Главные факторы (которых нет в коде совсем)

**Категория земли / ВРИ — важнейший фактор:**
| Категория | Относительная стоимость |
|-----------|------------------------|
| ИЖС | 1.0 (базовый) |
| ЛПХ в черте города | 0.5–0.7 |
| СНТ | 0.3–0.5 |
| Сельхоз | 0.01–0.02 |
- Перевод сельхоз → ИЖС: разница в цене до **130x**

**Коммуникации:**
- Полный набор (газ + вода + электро + канализация): +30–50% к цене
- Отсутствие коммуникаций: скидка ~20%
- Каждая коммуникация отдельно влияет, особенно газ

**Удалённость от города:**
- До 30 км от миллионника: ликвидно
- 18 км vs 49 км от города = цена **4.5x**
- >50 км: проблемный сегмент

**Транспортная доступность:**
- Асфальт: коэффициент ×1.4
- Грунтовка: коэффициент ×0.7

**Охранные зоны и сервитуты:**
- Охранные зоны ЛЭП/газопровода: –7–10% к стоимости
- Сервитут: обременение, снижает ликвидность

#### Сроки продажи участков ИЖС
| Ситуация | Срок |
|----------|------|
| С коммуникациями, <30 км от миллионника | 1 день – 1 месяц |
| Средний участок ~15–30 сот, <30 км | ~2 месяца |
| Без коммуникаций, >50 км | 6–24 месяца |
| Сельхоз, удалённый, без перспектив | >4 лет |

#### Дисконты на торгах (земля)
- ИЖС: 30–55% от рыночной цены
- Удалённые/сельхоз: 70–80%

#### Рекомендуемые топ-5 факторов для invest_score (земля)
1. Категория/ВРИ (25%)
2. Удалённость от города (25%)
3. Наличие коммуникаций (20%)
4. Дисконт к рынку (15%)
5. Транспортная доступность + охранные зоны (15%)

### ДОМА (ИЖС, коттеджи, таунхаусы)

**Дом vs квартира по ликвидности:**
- Квартира продаётся в 2–4 раза быстрее
- Квартиры: рост цены ~12%/год, дома ~7%/год (РФ, 2024)
- Дом быстрее окупается через аренду (123 мес. vs 175 мес. для квартиры)

**Факторы ликвидности дома:**
- Материал: кирпич > брус > каркас
- Год постройки: чем новее — тем ликвиднее
- Площадь: оптимум 80–150 м² (больше — уже узкий рынок)
- Коммуникации: критичнее, чем для квартиры (особенно центральная канализация)
- Участок: 6–20 соток оптимально

**Правовые риски домов:**
- Неузаконенные постройки/пристройки: риск предписания о сносе за счёт покупателя
- Прописанные несовершеннолетние: стоп-фактор
- Отказники от приватизации: стоп-фактор

### КОММЕРЧЕСКАЯ НЕДВИЖИМОСТЬ

#### Базовые формулы (нужны для расчёта)

```
Cap Rate = NOI / Рыночная стоимость × 100%
NOI = Годовая арендная выручка - Потери от вакансии (5–15%) - Операционные расходы
Стоимость объекта = NOI / Cap Rate
```

#### Нормальный Cap Rate для Перми (города-миллионники)
| Тип | Cap Rate |
|-----|----------|
| Стрит-ретейл (ПСН) | 10–13% |
| ТЦ с якорным арендатором | 12–15% |
| Офис класс А | 13–16% |
| Офис класс Б/В | 14–18% |
| Склад (современный) | 12–16% |
| Производство | 16–22%+ |

- Cap Rate < 8% → объект переоценён
- Cap Rate 10–14% → рыночная норма для регионов
- Cap Rate > 20% → высокий риск или скрытая проблема
- При ключевой ставке ЦБ 21%: Cap Rate < 12% не имеет смысла (депозит выгоднее)

#### Факторы, отсутствующие в текущем коде

**Наличие арендатора:**
- Действующий арендатор с договором = +30–60% к инвест. стоимости
- Сетевой арендатор (Магнит, DNS, Wildberries): прайм-цена
- Долгосрочный договор (5+ лет без права выхода) = +20% к цене

**Физические характеристики:**
- Высота потолков: ретейл мин. 3.5 м; склад 8–12 м = +40% к арендной ставке vs 4 м
- Отдельный вход: нет входа = –20–35% к цене
- Электромощность: >50 кВт = универсальность (общепит + магазин + офис)
- Парковка: норма 1 машиноместо на 30–50 м² торговли

**Ликвидный размер по типам:**
| Тип | Ликвидный размер |
|-----|-----------------|
| ПСН / стрит-ретейл | 50–300 м² |
| Офис | 100–500 м² |
| Склад | 500–5000 м² |
| Производство | от 1000 м² (низкая ликвидность всегда) |
| ТЦ | от 2000 м² (неликвид для розничного инвестора) |

#### Специфические риски коммерции

| Риск | Критичность | Описание |
|------|-------------|----------|
| Долги по КУ | ВЫСОКИЙ | Переходят к покупателю, часто накоплены на банкротных торгах |
| Самовольная постройка | КРИТИЧНО | Снос за счёт покупателя (ст. 222 ГК) |
| Нет разрешения на ввод | КРИТИЧНО | Нельзя легально эксплуатировать |
| Узкое назначение | ВЫСОКИЙ | Автосервис, автомойка — перепрофилирование = новое строительство |
| Незаконная перепланировка | ВЫСОКИЙ | Штраф + обязанность вернуть в исходное состояние |
| Концентрация: 1 арендатор | ВЫСОКИЙ | Уходит = нулевой доход |
| Обременение залогом банка | КРИТИЧНО | Может остаться после торгов — проверять ЕГРН |

#### Рекомендуемые веса для invest_score (коммерция)

| Блок | Вес |
|------|-----|
| Cap Rate (NOI/цена) | 25% |
| Локация и трафик | 20% |
| Арендный поток (есть/нет, тип, договор) | 20% |
| Физика объекта (состояние, высота, мощности) | 15% |
| Потенциал роста NOI | 10% |
| Рыночный контекст (вакансия, конкуренция) | 10% |

## Стоп-факторы (для всех типов)

Автоматический отказ независимо от остальных баллов:

| Стоп-фактор | Тип | Основание |
|-------------|-----|-----------|
| Отказник от приватизации | квартира/дом | Пожизненное право проживания, нельзя выселить (ст. 19 ФЗ-189) |
| Единственное жильё без ипотеки | квартира/дом | Не может продаваться в банкротстве по закону |
| Единственное ипотечное жильё (ФЗ-298, 2024) | квартира/дом | Суд может отменить торги |
| Несогласуемая перепланировка (снос несущих стен) | квартира | Покупатель не получит ипотеку |
| Самовольная постройка (ст. 222 ГК) | дом/коммерция | Снос за счёт покупателя |
| Аварийный дом | квартира/дом | Нет залоговой ценности, банки не кредитуют покупателя |
| МК + доли не выделены + нет согласия опеки | квартира | Высокий риск отмены сделки |
| Нет разрешения на ввод в эксплуатацию | коммерция | Нельзя легально эксплуатировать |
| Начальная цена выше рыночной | все | Нет смысла участвовать |
