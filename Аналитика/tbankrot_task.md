## Алгоритм сравнения

1. Загрузить наши лоты из Google Sheets (оба листа)
2. Загрузить лоты из Т-Банкрот (Пермь, недвижимость)
3. Матчинг:
   - **Точный**: по кадастровому номеру (если есть у обоих)
   - **Приблизительный**: по цене ± 5% + похожий адрес (первые слова)
4. Результат: три группы:
   - Есть у обоих → совпадение
   - Только в Т-Банкроте → **мы пропустили**
   - Только у нас → нет на Т-Банкроте (нормально)
