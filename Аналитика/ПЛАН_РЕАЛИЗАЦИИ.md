# Estate Invest Analytics — План реализации

## Блоки (агенты)

---

### Блок 1: DATA LOADER — Загрузчик данных
**Задача:** Подключиться к Google Sheets и загрузить все данные в чистом виде

Что делает:
- Подключение к Google Sheets API (read-only)
- Загрузка 6 ключевых листов: Общая база, Инвесторы, Объекты, Свод инвесторы, Свод объекты, Свод объекты+инвесторы
- Парсинг: числа из "5 228 170,99р." → 5228170.99, даты из "01.07.2022" → datetime
- Обработка мультивалютности: $, €, USDT, Р. + курсы из строк 1-3
- Кэширование (не загружать 76K строк каждый запрос)

Результат: чистые pandas DataFrames, готовые для анализа

---

### Блок 2: ANALYTICS ENGINE — Аналитический движок
**Задача:** Считать все метрики из сырых данных

Модули:
- **investor_analytics** — по инвестору:
  - Сколько вложил (по валютам и итого в рублях)
  - Сколько получил (выплата тела + выплата %)
  - Остаток (свободный + в объектах)
  - В каких объектах участвует, доля в каждом
  - История транзакций

- **object_analytics** — по объекту:
  - P&L: цена покупки + доп. траты → цена продажи → чистая прибыль → ROI
  - Статус сбора: собрано / осталось / % заполнения
  - Список инвесторов с долями и выплатами
  - Сроки: план vs факт, скорость реализации

- **portfolio_analytics** — общая картина:
  - Всего привлечено / выплачено / в работе
  - Количество инвесторов и объектов
  - Рейтинг объектов по ROI
  - Долговая нагрузка (сколько нужно вернуть)

Результат: функции, которые принимают DataFrames и возвращают метрики

---

### Блок 3: ADMIN DASHBOARD — Панель управляющего
**Задача:** Веб-интерфейс с полной картиной для управляющего

Страницы:
- **Главная** — ключевые цифры + графики
- **Инвесторы** — таблица с поиском → карточка инвестора
- **Объекты** — таблица с фильтрами → карточка объекта с P&L
- **Сравнение объектов** — рейтинг, графики, фильтры по типу/стадии
- **Выплаты** — кому выплачено / кому должны

Защита: пароль для входа

Результат: работающий Streamlit-дашборд для управляющего

---

### Блок 4: INVESTOR PORTAL — Личный кабинет инвестора
**Задача:** Инвестор видит только свои данные

Страницы:
- Мои вложения и объекты
- Мои выплаты (история)
- Мой баланс по валютам

Авторизация: код/ссылка для каждого инвестора (по email или ID)

Результат: инвестор заходит и видит свой портфель

---

### Блок 5: DEPLOY — Деплой и запуск
**Задача:** Запустить всё в продакшен

- Настройка Google Sheets API credentials (сервисный аккаунт)
- Деплой на Streamlit Cloud или VPS
- Настройка домена (analytics.estateinvest.online)
- Тестирование на реальных данных

Результат: рабочий инструмент, доступный по ссылке

---
