# Монитор торгов — Дорожная карта

Система мониторинга лотов: torgi.gov.ru + ЕФРСБ → Supabase → lots-monitor.html

---

## Готово ✅

- Парсинг лотов с torgi.gov.ru (cron каждые 4 часа на VPS)
- Парсинг лотов с ЕФРСБ (cron каждые 4 часа на VPS)
- Запись в Google Sheets (лист "Лоты")
- Уведомления в Telegram (@topparsing)
- PropertyCard — карточка объекта с блоками (квартира, земля, коммерция, дом)
- Обогащение координатами через DaData
- Рыночная оценка (аналоги с Авито/Циан через torgi_monitor)
- Кадастральные данные через НСПД
- Парсинг должника из описания лота (имя, ИНН, тип)
- Скоринг v1 — invest_score, risk_score, liquidity_score, рекомендация, прибыль, ROI
- Фронтенд lots-monitor.html — карточки, таблица, Яндекс Карты, фильтры, модальное окно
