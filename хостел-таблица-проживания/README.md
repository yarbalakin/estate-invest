# Хостел — Автогенерация таблицы проживания

## Установка в Google Sheets (один раз)

1. Открой Google Sheets с данными хостела
2. **Extensions → Apps Script**
3. Удали весь стандартный код в `Code.gs`
4. Вставь содержимое файла `Code.gs` из этой папки
5. Нажми **+ (New file) → HTML**, назови файл `Dialog` (без расширения)
6. Удали стандартный код, вставь содержимое `Dialog.html`
7. Сохрани (Ctrl+S)
8. Закрой вкладку Apps Script
9. Перезагрузи Google Sheets — появится меню **Хостел**

## Использование

1. В Google Sheets: меню **Хостел → Сформировать таблицу проживания**
2. При первом запуске Google попросит разрешения — нажми **Разрешить**
3. В диалоге выбери контрагента и месяц
4. Нажми **Сформировать таблицу**
5. Через несколько секунд появится новый лист с таблицей

## Что генерирует

- Лист с названием `Таблица — Январь 2026 — ГеоИнвест`
- Список сотрудников с отметками по дням месяца (кто когда жил)
- Итого: количество человеко-дней и сумма
- Строки для подписей обеих сторон

## Настройки (в начале Code.gs)

```js
var LOCATION  = 'г. Краснокамск'; // объект
var HOST_NAME = 'ИП Мудров И.Ю.'; // принимающая сторона
```

## Как работает фильтр по компании

Скрипт ищет введённое название как подстроку в колонке C (Название компании, номер комнаты).
Пример: если ввести `ГеоИнвест` — найдёт все строки где есть слово ГеоИнвест.
Регистр не важен.
