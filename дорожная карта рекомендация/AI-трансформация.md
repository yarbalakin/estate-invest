# Estate Invest — Дорожная карта AI-трансформации

> Цель: максимальная автоматизация операций через AI. Путь от ручного управления к AI-управляемой компании.
> Дата: 26.02.2026

---

## Концепция

```
Сейчас:  Люди делают работу → иногда используют инструменты
Цель:    AI делает работу → люди принимают решения
```

8 отделов. Каждый получает свой AI-слой. Приоритет: аналитика → реализация → маркетинг → поддержка → остальные.

---

## Фаза 1 — АНАЛИТИКА (1–3 мес.) 🔥 Старт здесь

### Что сейчас
Ручной мониторинг торговых площадок. Аналитики смотрят лоты вручную, делают финмодели в таблицах.

### Что строим: Ежедневный AI-парсер объектов

**Поток:**
```
Торговые площадки (все РФ)
    → Парсинг лотов (n8n)
        → AI-анализ каждого лота (OpenRouter)
            → Скоринг: горячий / интересный / пропустить
                → Telegram-отчёт партнёрам (топ-5 дня)
```

**Источники для парсинга:**
- tbankrot.ru — банкротные торги (уже есть в Apps Script)
- m-ets.ru — муниципальные торги
- torgi.gov.ru — федеральные торги
- rosimushchestvo.ru — аресты
- Авито / ЦИАН — рыночные цены (для сравнения)

**AI-анализ каждого лота:**
- Тип объекта, регион, начальная цена
- Рыночная цена (через Авито-парсер)
- Потенциал скидки от рынка (%)
- Ожидаемый ROI-диапазон (исходя из исторических данных компании)
- Риски: обременения, тип торгов, сроки
- Итог: горячий / интересный / пропустить + причина

**Telegram-отчёт (ежедневно 08:00 МСК):**
```
🔥 Топ-5 объектов за 25.02.2026

1. Краснодар, ЗУ 12 сот. | 800К → рынок 2.1М | ROI ~40% | 🟢 Горячий
   Риски: нет обременений, задаток 80К, торги 05.03
   ...
```

**Расширение до всей России:**
- Фаза 1: Пермь + Калининград + Самара (уже работаем)
- Фаза 2: + Краснодар + Нижний Новгород + Екатеринбург
- Фаза 3: все регионы РФ (100+ площадок)

**Стек:** n8n + OpenRouter (Llama/GPT) + Telegram + Google Sheets (история лотов)

## Фаза 2 — РЕАЛИЗАЦИЯ ОБЪЕКТОВ (2–4 мес.)

### Что сейчас
16 таблиц Google Sheets, 117 индивидуальных касс. Данные разбросаны. Нет единой картины.

### Что строим: Единая панель управления бизнесом

**2.1 Единая база данных объектов**
- Все 118+ объектов в одном месте (1С или custom DB)
- Каждый объект: статус, финансы, документы, таймлайн, ответственные
- Автосинхронизация: 1С ↔ ЛК инвестора ↔ дашборд партнёров

**2.2 Канбан объектов с автопереходами**
```
Торги → Выкуп → Документы → Ремонт → Продажа → Закрыт
```
- При смене статуса → автоуведомление ответственному
- При заходе в «Ремонт» → автозапрос бюджета и сроков от Ивана
- При заходе в «Продажа» → автогенерация объявления (см. ниже)

**2.3 AI-генерация объявлений**
- Из карточки объекта (адрес, тип, площадь, фото) → AI пишет объявление для Авито
- Варианты заголовков (A/B тест)
- Оптимальная цена: AI смотрит на аналоги в радиусе 5 км

**2.4 AI-оценщик**
- Загружаешь параметры объекта → AI даёт диапазон рыночной цены
- Источники: парсинг Авито + история продаж компании (120+ объектов)
- Точность растёт с каждой новой сделкой (обучение на своих данных)

**2.5 Дашборд партнёров (реальное время)**
- AUM, активные объекты по стадиям, денежный поток
- Прогресс к 24 объектам в год
- Топ-5 объектов по ROI, топ-5 зависших

**Стек:** n8n + 1С + OpenRouter + Google Sheets + HTML-дашборд

## Фаза 3 — МАРКЕТИНГ (3–5 мес.) — AI полностью

### Что сейчас
Нет выделенного маркетолога. Контент делается вручную Глебом. Нет платных каналов.

### Что строим: AI-маркетинговая машина

**3.1 AI-контент из данных (автоматически)**
- Закрылся объект → AI автоматически пишет кейс: «Купили за X, продали за Y, ROI Z%»
- Форматы: пост в Telegram, карточка для VK, короткий текст для Reels
- Глеб только проверяет и публикует (10 мин вместо 2 часов)

**3.2 Автопостинг**
- n8n: контент-план → автопубликация в Telegram + VK + Дзен
- Оптимальное время публикации (AI определяет по вовлечённости)
- A/B тестирование заголовков

**3.3 AI рекламные кампании**
- VK Ads + Яндекс.Директ: AI генерирует тексты объявлений
- Look-alike аудитории от базы 900 инвесторов
- Автоматическая оптимизация ставок (пока через агента)

**3.4 AI-воронка прогрева**
- Новый лид из рекламы → автосерия в Salebot (7 дней)
- День 1: «Как мы нашли объект за 800К и продали за 2.1М»
- День 3: «Почему торги по банкротству — это законно и безопасно»
- День 5: Текущие объекты с ROI
- День 7: Приглашение на созвон
- Всё генерирует AI, Глеб настраивает один раз

**3.5 AI-аналитика маркетинга**
- Откуда пришли инвесторы (UTM → CRM)
- Какой контент конвертирует лучше
- CAC по каналам в реальном времени

**Стек:** n8n + OpenRouter + Salebot + VK API + Telegram API + Яндекс.Метрика

## Фаза 4 — ПОДДЕРЖКА ИНВЕСТОРОВ (4–6 мес.)

### Что сейчас
3 человека отвечают на одни и те же вопросы вручную. Salebot закрывает часть, но без AI.

### Что строим: AI первая линия поддержки

**4.1 AI-бот первой линии**
- Апгрейд Salebot: подключить AI к свободному вводу
- 80% вопросов инвесторов типовые → AI отвечает мгновенно
- «Когда выплата по ИП №45?» → AI смотрит в базу и отвечает
- «Какая у меня доходность?» → AI считает из данных ЛК

**4.2 Умные уведомления**
- Объект перешёл в стадию «Продажа» → автоуведомление инвесторам
- Новый объект открыт → персональный подбор («Вам подойдёт, вы вкладывали в похожее»)

**4.4 Onboarding-автоматизация**
- Регистрация → автоматически: договор (SignEasy) → инструкция → приглашение в каналы → welcome-звонок
- Сейчас это делают вручную 3 человека

**Стек:** Salebot + OpenRouter + n8n + ЛК API

## Фаза 5 — ФИНАНСЫ И СТАТИСТИКА (5–7 мес.)

### Что сейчас
Александра вручную ведёт статистику. 76 000 строк в таблицах. Отчёты делаются руками.

### Что строим: Автоматическая финансовая отчётность

**5.1 Автоотчёты**
- Еженедельный отчёт партнёрам: AUM, новые инвестиции, выплаты, ТОП объектов
- Ежемесячный отчёт инвесторам: их портфель, доходность, прогноз
- Всё генерирует AI из данных 1С, Александра только проверяет

**5.3 Дашборд реального времени**
- Капитал под управлением (AUM)
- Денежный поток: приток инвестиций vs выплаты
- ROI по типам объектов и регионам
- Прогресс к годовым целям

Это готово уже - в пульсе. 

**5.4 Автоматический расчёт выплат**
- Объект закрыт → AI автоматически рассчитывает долю каждого инвестора
- Формирует реестр выплат → на проверку партнёру → отправка в банк

**Стек:** 1С + n8n + Google Sheets → HTML-дашборд + Telegram

## Фаза 6 — ЮРИДИЧЕСКИЙ ОТДЕЛ (6–8 мес.)

### Что сейчас
Галина вручную проверяет документы по объектам. Шаблоны договоров заполняются руками.

### Что строим: AI-юридический ассистент

**6.1 AI-проверка лота перед выкупом**
- Загружаешь документы лота (PDF) → AI анализирует:
  - Обременения, аресты, залоги
  - Правовые риски (самозахват, споры, наследство)
  - Соответствие заявленным характеристикам
- Вывод: «Чисто» / «Есть риски: [список]» / «Стоп»

**6.2 Автозаполнение договоров**
- Выбрал инвестора + объект + сумму → AI заполняет инвестиционный договор
- Отправка через SignEasy автоматически
- Хранение подписанных договоров в ЛК

**6.3 Мониторинг изменений законодательства**
- AI еженедельно мониторит изменения ФЗ-259, ФЗ-335, налоговое законодательство, по ФЗ-127
- Уведомление партнёрам если что-то меняется

**6.4 AI Due Diligence**
- Кадастровая проверка (через API Росреестра)
- Автоматическая проверка обременений
- Анализ судебных решений по адресу

**Стек:** n8n + OpenRouter + Росреестр API + SignEasy API

## Фаза 7 — КОНТЕНТ (параллельно с Фазой 3)

### Что сейчас
Глеб и Алёна вручную ведут Telegram, VK, Дзен. Контент-план в Google Sheets.

### Что строим: AI-контент-студия

**7.1 Фабрика кейсов**
- База: 120+ закрытых объектов с данными (купили/продали/ROI/срок)
- AI генерирует кейс из данных: история, цифры, уроки
- Форматы: текст (Telegram/VK), сценарий видео, карусель

**7.2 AI редактор**
- Глеб набрасывает мысль → AI разворачивает в пост
- Проверка тона (не слишком рекламно, доверительно)
- SEO-оптимизация для Дзена

**7.3 Видео-контент**
- AI пишет сценарий для YouTube-видео из данных объекта
- Скрипт для Reels (60 сек, крючок + история + CTA)
- Субтитры автоматически (Whisper API)

**7.4 Мультиплатформенность**
- Один кейс → AI адаптирует под 5 форматов: Telegram пост, VK пост, Reels сценарий, Дзен статья, email рассылка
- Глеб проверяет, n8n публикует по расписанию

7.5 Генераци контента из новостей

**Стек:** n8n + OpenRouter + Telegram API + VK API + Дзен API + Whisper

```
→ Топ-10 горячих лотов по всей России с AI-анализом

→ ИП №119 перешёл в "Продажа" → объявление уже написано AI

→ Проверяет автоотчёт за неделю вместо составления

```
