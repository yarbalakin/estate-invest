Фаза 3
Маркетинг — AI-машина полностью
3–5 мес. • Стек: n8n + OpenRouter + Salebot + VK API + Telegram API + Яндекс.Метрика
Сейчас
  • Нет выделенного маркетолога
  • Контент делается вручную Глебом
  • Нет платных каналов привлечения
  • Рост — органика и сарафан, потолок близко
После
  • AI генерирует кейсы автоматически из данных объектов
  • Автопостинг: Telegram + VK + Дзен по расписанию
  • AI-воронка прогрева новых лидов (7 дней)
  • CAC < 5К, 3 кейса в неделю без усилий
3.1 AI-контент из данных
  • Закрылся объект → AI пишет кейс: «Купили за X, продали за Y, ROI Z%»
  • Форматы: пост Telegram, карточка VK, текст для Reels
  • Глеб только проверяет (10 мин вместо 2 часов)
  • 120+ закрытых объектов = контент на год вперёд
3.2 Автопостинг
  • n8n публикует по расписанию во все каналы
  • Оптимальное время публикации (AI по вовлечённости)
  • A/B тестирование заголовков автоматически
3.3 AI рекламные кампании
  • VK Ads + Яндекс.Директ: AI генерирует тексты
  • Look-alike аудитории от базы 900 инвесторов
  • Автоматическая оптимизация ставок
3.4 AI-воронка прогрева (7 дней)
  • День 1: «Купили за 800К, продали за 2.1М» — кейс
  • День 3: «Почему торги — это законно и безопасно»
  • День 5: Текущие объекты с ROI
  • День 7: Приглашение на созвон
  • Всё генерирует AI, Глеб настраивает один раз
3.5 AI-аналитика маркетинга
  • Откуда пришли инвесторы (UTM → CRM)
  • Какой контент конвертирует лучше
  • CAC по каналам в реальном времени
Стек: n8n • OpenRouter • Salebot • VK API • Telegram API • Яндекс.Директ • Яндекс.Метрика

5.2 Дашборд реального времени
  • AUM (капитал под управлением)
  • Денежный поток: приток инвестиций vs выплаты
  • ROI по типам объектов и регионам
  • Прогресс к годовым целям  Пульс уже работает ✓
5.3 Автоматический расчёт выплат
  • Объект закрыт → AI рассчитывает долю каждого инвестора
  • Формирует реестр выплат
  • На проверку партнёру → отправка в банк
Стек: 1С • n8n • Google Sheets • HTML-дашборд • Telegram
Фаза 6
Юридический — AI-ассистент
6–8 мес. • Стек: n8n + OpenRouter + Росреестр API + SignEasy API
Сейчас
  • Галина вручную проверяет документы объектов
  • Договоры заполняются руками по шаблону
  • Due diligence занимает 1–2 дня
После
  • AI анализирует документы лота за 10 минут
  • Договоры автозаполняются и уходят через SignEasy
  • Мониторинг изменений законодательства автоматически
6.1 AI-проверка лота
  • Загружаешь документы (PDF) → AI анализирует
  • Обременения, аресты, залоги, правовые риски
  • Самозахват, споры, наследство, соответствие характеристикам
  • Вывод: «Чисто» / «Риски: [список]» / «Стоп»
6.2 Автозаполнение договоров
  • Выбрал инвестора + объект + сумму → договор готов
  • Отправка через SignEasy автоматически
  • Подписанные договоры хранятся в ЛК
6.3 Мониторинг законов + Due Diligence
  • AI еженедельно мониторит изменения ФЗ-259, ФЗ-127
  • Уведомление партнёрам при изменениях
  • Кадастровая проверка через API Росреестра
  • Анализ судебных решений по адресу
Стек: n8n • OpenRouter • Росреестр API • SignEasy API
Фаза 7
Контент — AI-студия (параллельно с Ф3)
Параллельно с Фазой 3 • Стек: n8n + OpenRouter + Telegram API + VK API + Дзен + Whisper
Сейчас
  • Глеб вручную ведёт Telegram, VK, Дзен
  • Контент-план в Google Sheets
  • Один человек на все каналы
После
  • Один кейс → 5 форматов автоматически
  • 120+ объектов = бесконечный поток историй
  • Глеб управляет контентом, не пишет его
7.1 Фабрика кейсов
  • База: 120+ объектов с данными (купили / продали / ROI / срок)
  • AI генерирует кейс из данных: история, цифры, уроки
  • Форматы: текст (Telegram/VK), сценарий видео, карусель
7.2 AI-редактор
  • Глеб набрасывает мысль → AI разворачивает в пост
  • Проверка тона: не рекламно, доверительно
  • SEO-оптимизация для Дзена автоматически
7.3 Видео-контент
  • AI пишет сценарий YouTube-видео из данных объекта
  • Скрипт для Reels (60 сек: крючок + история + CTA)
  • Субтитры автоматически через Whisper API
7.4 Мультиплатформенность
  • Один кейс → Telegram, VK, Reels, Дзен статья, email
  • Глеб проверяет, n8n публикует по расписанию
  • 5 форматов из одного источника без дополнительного труда
Стек: n8n • OpenRouter • Telegram API • VK API • Дзен API • Whisper API

8.1 AI-советник в ЛК
  • Анализирует портфель и даёт рекомендации
  • «У вас 300К свободного баланса. Открыт ИП №119, прогноз 28%»

8.2 Умный поиск объектов
  • Фильтры: регион, тип, ROI, срок, сумма
  • AI-подбор: «Вы вкладывали в ИП №40 (земля, 34%),
    вам может подойти ИП №115»
8.3 AI-ассистент разработки
  • Claude Code / Cursor ускоряет разработку ЛК
  • Автотесты через AI
  • Code review через AI
Стек: React (ЛК) • n8n • OpenRouter • Claude API
