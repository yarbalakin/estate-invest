## Почему разделяем

Сейчас на одних людей навалены две принципиально разных работы:

| Характеристика | Реализация (оформление + улучшение) | Продажи |
|---------------|-------------------------------------|---------|
| **Тип работы** | Юридическая + физическая (стройка) | Коммерческая |
| **Навыки** | Юриспруденция, кадастр, стройка, снабжение | Переговоры, маркетинг, клиентский сервис |
| **Темп** | Длинные циклы (1-6 мес.), зависимость от госорганов | Быстрые итерации, ежедневные контакты |
| **Мотивация** | Результат = готовый объект | Результат = закрытая сделка |
| **Метрики** | Сроки, бюджет, качество | Конверсия, скорость, количество |

Смешивать их = перегруз + конфликт приоритетов (что важнее — доделать ремонт или поехать на показ?).

## Проблема: улучшение объектов — кто делает?

Улучшение объектов (п.4 в Реализации) — это **самая трудоёмкая часть**. Сейчас Иван делает это на энтузиазме. Нужно решить, как это системно организовать.

### Типы улучшений по сложности

| Тип | Примеры | Сложность | Кто может делать |
|-----|---------|-----------|-----------------|
| **Простые** | Уборка, мелкий ремонт, фотосъёмка, вывоз мусора | Низкая | Риэлтор, помощник |
| **Средние** | Косметический ремонт, покраска, замена сантехники | Средняя | Прораб/куратор + бригада |
| **Сложные** | Межевание, раздел помещений, перевод ВРИ, капремонт | Высокая | Иван + кадастровый инженер + юрист |
| **Юридические** | Снятие обременений, регистрация, работа с конкурсным | Высокая | Иван + Галина (юрист) |

### Варианты организации

#### Вариант A: Риэлтор-куратор (один человек от начала до конца)

```
Объект закреплён за одним риэлтором: улучшение → продажа
```

**Плюсы**:
- Максимальная мотивация: быстрее улучшит = быстрее продаст = быстрее комиссия
- Один ответственный, нет передач между отделами
- Глубокое знание объекта

**Минусы**:
- Таких людей мало (нужен и строитель, и продажник)
- Риэлтор на стройке = не продаёт другие объекты
- Один человек не может параллелить 5+ объектов с ремонтом

**Когда подходит**: для простых объектов (земля без ремонта, чистая квартира)

#### Вариант B: Выделенный прораб/куратор объектов

```
Прораб: улучшение → передача → Риэлтор: продажа
```

**Плюсы**:
- Специализация: каждый делает то, что умеет
- Можно нанять за фиксированный % от бюджета ремонта или оклад 40-60К
- Снимает 80% нагрузки с Ивана

**Минусы**:
- Дополнительные расходы на ФОТ
- Нужно найти надёжного человека (доступ к объектам, деньги на материалы)

**Когда подходит**: при 3+ объектах в ремонте одновременно

#### Вариант C: Гибридная модель (РЕКОМЕНДУЕТСЯ)

```
Простые улучшения → Наталья
Сложные улучшения → Иван + прораб (когда наймём)
Юридические вопросы → Иван + Галина
```

**Плюсы**:
- Реалистично для текущего масштаба
- Постепенный переход: сначала без найма, потом прораб
- Риэлторы учатся работать с простыми объектами — растут

**Минусы**:
- Нужно чёткое разграничение "простой/сложный"

**Как разграничить**:
- **Простой объект**: не требует ремонта > 200К, не требует юридических действий (межевание, раздел, ВРИ)
- **Сложный объект**: ремонт > 200К, или межевание, или раздел, или перевод ВРИ, или капремонт

## Математика: стоит ли платить за куратора объектов

**Сейчас** (Иван на энтузиазме):
- Иван тратит ~60% времени на курирование объектов
- Это время можно было бы тратить на стратегию, новые объекты, переговоры
- Риск: если Иван заболеет/уедет — всё встаёт

**С куратором** (40-60К/мес):
- Иван освобождает ~60% времени → может взять больше объектов или заняться масштабированием
- Куратор ведёт 5-8 объектов параллельно
- Снабжение, контроль бригад, фотоотчёты — снято с Ивана

**Расчёт окупаемости**:
- Зарплата куратора: 50К/мес
- Если сокращает срок реализации объектов хотя бы на 2 недели (быстрее продаём → быстрее прибыль)
- При средней прибыли ~1 млн с объекта и 2 объекта/мес → ускорение на 2 недели = экономия процентов на заёмные средства + быстрее оборот
- **Вывод**: окупается при 3+ объектах в одновременной подготовке

- Объекты с незавершённым ремонтом
- Объекты с неснятыми обременениями
- Объекты без зарегистрированного права

## Третья функция: Содержание имущества (управление)

### Проблема
Пока объект в собственности (а это 3-12 месяцев), его нужно **содержать**:
- Оплата коммунальных услуг
- Контроль коммунальных платежей (чтобы не копился долг)
- Косить борщевик (земельные участки — штрафы за борщевик до 100К)
- Латать крыши, следить за состоянием
- Охрана (для коммерческих / пустующих)
- Зимой: чистка снега, отопление (чтобы трубы не лопнули)
- Страхование (если оформлено)

Это **рутинная операционная работа** — она не имеет начала и конца, идёт пока объект в собственности. Принципиально отличается от проектной работы (реализация) и коммерческой (продажи).

| Вариант | Описание | Стоимость | Плюсы | Минусы |
